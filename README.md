# Virat Kohli Batting Analysis (2008–2018)

## Project Description

This project focuses on analyzing Virat Kohli’s batting performance from 2008 to 2018 using cricket statistics and exploratory data analysis techniques. The analysis highlights his transformation as a batsman after 2012, where improvements in consistency, match temperament, shot selection, and run-scoring ability established him as one of the most dependable batsmen in modern cricket. The project uses visual storytelling and statistical analysis to study his growth and performance trends over the years.


## Objectives

* Analyze Virat Kohli’s batting performance from 2008–2018
* Study his transformation and improvement after 2012
* Identify yearly scoring and consistency trends
* Compare performances against different opposition teams
* Understand batting contribution through data visualization
* Generate sports analytics insights using cricket data

## Sports Insights

* Virat Kohli’s batting performance improved significantly after 2012 due to increased consistency, better shot selection, and stronger match temperament
* The analysis reflects his transformation from a young aggressive batsman into a mature and dependable run-scorer
* Certain opposition teams witnessed notably stronger performances from Kohli
* Year-wise trends demonstrate continuous growth in performance and match impact
* Visualizations highlight how Kohli became one of the most dominant batsmen during 2012–2018
